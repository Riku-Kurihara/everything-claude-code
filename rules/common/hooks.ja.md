# フックシステム

## フックタイプ

- **PreToolUse**：ツール実行前（検証、パラメータ変更）
- **PostToolUse**：ツール実行後（自動フォーマット、チェック）
- **Stop**：セッション終了時（最終検証）

## 自動許可

慎重に使用：
- 信頼できる、明確に定義されたプランのために有効化
- 探索的作業では無効化
- 決して dangerously-skip-permissions フラグを使用しない
- `~/.claude.json` の `allowedTools` を代わりに設定

## TodoWrite ベストプラクティス

TodoWrite ツールを使用して：
- 複数ステップのタスクの進行状況を追跡
- 指示の理解を確認
- リアルタイムの方向付けを有効化
- 細かい実装ステップを表示

Todo リストは以下を明らかにします：
- 順序が正しくないステップ
- 欠落しているアイテム
- 不要な追加アイテム
- 粒度が間違っている
- 要件の誤解
