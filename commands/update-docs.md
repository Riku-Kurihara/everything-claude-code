# ドキュメントを更新

信頼できるソースからドキュメントを同期:

1. package.json スクリプトセクションを読む
   - スクリプト参照テーブルを生成
   - コメントから説明を含める

2. .env.example を読む
   - すべての環境変数を抽出
   - 目的とフォーマットをドキュメント

3. docs/CONTRIB.md を生成:
   - 開発ワークフロー
   - 利用可能なスクリプト
   - 環境設定
   - テスト手順

4. docs/RUNBOOK.md を生成:
   - デプロイメント手順
   - 監視とアラート
   - 一般的な問題と修正
   - ロールバック手順

5. 廃止されたドキュメントを特定:
   - 90+ 日間変更されていないドキュメントを検索
   - 手動レビュー用にリスト

6. 差分サマリーを表示

信頼できるソース: package.json と .env.example
